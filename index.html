Programacion

La programación es el proceso mediante el cual a través de codigo se le dice a la computadora que hacer tomando en cuenta el orden para lograr un objetivo o solucionar un problema

VARIABLES

  • let: variables cambiantes
  • const: variables fijas

TIPOS DE DATOS

  • Number: integer or float/double
  • String
  • Boolean: Yes/No

Operadores

Los operadores son simbolos que le dicen a la computadora que operación matematica, relacion, logica realizar

  • Operadores aritmeticos: compuesto por +,-,*,/

    • console.log(number1+number2);

  • Operadores de asignación: compuesto por =, asigna valor a una variable, puede usarse let para una variable movil o const para una variable fija

    • let nombre = 'Diego';

    const nombre2 = 'Andre';

  • Operadores Comparacion: compuesto por >,<,>=,<=/

    • console.log(number1>number2);

  • Operadores Lógicos: compuesto por &&,||

    • const fiestaEntra = edad1>=mayoriaEdad || genero =='F';
